HyprVideo

Per-workspace video wallpapers with pseudo-tiling window management

Preview

Note: Not recommended for low-spec systems. 16GB RAM minimum is strongly advised.

Quick Start

git clone https://github.com/XclusivVv/hyprland-video-wallpapers.git
cd hyprland-video-wallpapers

chmod +x install.sh uninstall.sh
./install.sh

Features

  • Interactive setup wizard that guides you through installation.
  • Dependency checker for mpv, jq, socat (installs automatically on Arch).
  • Optional video optimizer for compressing heavy wallpapers.
  • Interactive video mapping for assigning videos to workspaces 1–10.
  • Hyprland auto-configuration with safe prompts for modifying configs.
  • Togglefloating conflict detection with automatic fix.
  • Complete uninstall script that reverts all changes.

What the Installer Does

  • Checks for hyprctl.
  • Installs mpv, socat, jq on Arch systems.
  • Configures layout gaps (top gap + gap size).
  • Lets you pick a video directory and map videos to workspaces.
  • Installs the main script:
    • ~/.local/bin/hyprland-video-wallpapers.sh
  • Creates user configuration:
    • ~/.config/hyprland-video-wallpapers/config.conf
    • ~/.config/hyprland-video-wallpapers/hyprland-video-wallpapers.conf
  • Offers to patch ~/.config/hypr/hyprland.conf:
    • Add source line
    • Disable conflicting togglefloating keybind

Hyprland Autostart

Add this to your Hyprland config:

exec-once = ~/.local/bin/hyprland-video-wallpapers.sh

Configuration File

Your complete settings are saved at:

~/.config/hyprland-video-wallpapers/config.conf

You can edit it manually — or simply rerun ./install.sh.

Uninstallation

./uninstall.sh

The uninstaller will:

  • Stop all mpv wallpaper processes
  • Remove exec-once from Hyprland
  • Remove sourced config lines
  • Restore togglefloating if it was disabled
  • Delete all installed files + optimizer tool

How It Works

  • Each workspace runs its own dedicated MPV instance.
  • The MPV window becomes the master window on its workspace.
  • Videos:
    • Play when you enter their workspace
    • Pause when you leave
  • All non-MPV windows are floating by rule.
  • The script listens for window events and arranges floating apps in a pseudo-tiling layout.
